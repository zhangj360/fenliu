针对bigo，meoo，momo直播做的分流规则


小火箭配置文件下载：
https://raw.githubusercontent.com/zhangj360/fenliu/refs/heads/main/Shadowrocket.conf


小火箭使用方法：打开小火箭，点击最下面配置选项，点击右上角+号，添加上面的链接，下载后点击启用。


karing分流规则文件下载：
https://raw.githubusercontent.com/zhangj360/fenliu/refs/heads/main/diversion_rules_custom.json


karing使用方法：打开karing,进入分流选项卡，进入分流规则，点击右上角，在自定义分流组后面点击，选择导入，选择diversion_rules_custom.json文件导入即可！
